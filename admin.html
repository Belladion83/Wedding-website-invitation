Thiết lập Google Sheet / Apps Script
1) Tạo Google Sheet mới. 2) Mở Extensions → Apps Script. 3) Dán file google-apps-script.gs. 4) Deploy Web App. 5) Dán URL Web App vào ô dưới đây. Lưu ý: để mọi thiết bị đều tự đọc dữ liệu cloud, URL này cần được cấu hình một lần trong file js/config.js hoặc được tôi cập nhật vào source sau khi bạn gửi URL.
Nút Lưu bản nháp chỉ lưu để bạn chỉnh thử trên thiết bị hiện tại. Nút Lưu sẽ publish lên Google Sheet/Google Drive để mọi thiết bị thấy thay đổi. Nếu chưa có Google Apps Script URL thì không thể đồng bộ toàn bộ thiết bị.
