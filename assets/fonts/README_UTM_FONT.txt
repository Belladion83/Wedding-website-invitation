Đặt file font UTM Edwardian Script có hỗ trợ tiếng Việt vào thư mục này.
Các tên file được CSS hỗ trợ sẵn:
- UTM-Edwardian-Script.woff2
- UTM-Edwardian-Script.otf
- UTM-Edwardian-Script.ttf

Khuyến nghị dùng WOFF2 cho web. Nếu chữ Trần/Nguyệt vẫn lỗi sau khi đặt font, font đó chưa có glyph tiếng Việt đầy đủ; cần dùng bản UTM Edwardian Script Việt hóa.
